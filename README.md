## Bakery Shop Website

## Overview

The Bakery Shop Website is a modern and responsive website designed to showcase a bakery's products, services, and contact information. The website includes the menu of items in the shop, contact info, stuning UI.

## Features

* Responsive design for desktop, tablet, and mobile devices.
* Home page with bakery products.
* Products menu displaying different types of breads with its price.
* About Us section introducing the ethics of the shop.
* Contact section with adress, phone number, email, and social media links.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Usage

Developed as a bakery shop website project by enhancing the marketing startegy of the shop.
